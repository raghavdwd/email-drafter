## Available Placeholders

Use these placeholders in your email templates:

| Placeholder                      | Description                                    | Example                |
| -------------------------------- | ---------------------------------------------- | ---------------------- |
| `{{First Name}}`                 | Contact person's name                          | Sergio                 |
| `{{Company Name}}`               | Client's company name                          | Smart LED Inc.         |
| `{{Website}}`                    | Client's website URL                           | smartledinc.com        |
| `{{Client Traffic}}`             | Client's monthly traffic                       | 453                    |
| `{{Competitor Business Name 1}}` | First competitor's name                        | Sonneman Light         |
| `{{Competitor Website 1}}`       | First competitor's URL                         | sonnemanlight.com      |
| `{{Competitor Traffic 1}}`       | First competitor's traffic                     | 1000                   |
| `{{Competitor Business Name 2}}` | Second competitor's name                       | -                      |
| `{{Competitor Website 2}}`       | Second competitor's URL                        | -                      |
| `{{Competitor Traffic 2}}`       | Second competitor's traffic                    | -                      |
| `{{Client Screenshot}}`          | Client screenshot URL (auto-embedded as image) | -                      |
| `{{Competitor Screenshot}}`      | Competitor screenshot URL (auto-embedded)      | -                      |
| `{{Email}}`                      | Contact email                                  | sergio@smartledinc.com |

## Tips for Writing Effective Templates

1. **Personalization**: Always use `{{First Name}}` and `{{Company Name}}` to make emails feel personal
2. **Data-Driven**: Reference specific numbers (`{{Client Traffic}}`, `{{Competitor Traffic 1}}`) to show you've done research
3. **Visual Proof**: Include screenshots using `{{Client Screenshot}}` and `{{Competitor Screenshot}}` - they'll be automatically embedded as images
4. **Multiple Competitors**: Use both competitor fields when available to show comprehensive analysis
5. **Clear CTA**: Always include a clear call-to-action (schedule a call, reply for more info, etc.)
6. **Keep It Concise**: Shorter emails typically get better response rates

## How Screenshots Work

When you include screenshot placeholders like `{{Client Screenshot}}` or `{{Competitor Screenshot}}`, the system will:

1. Convert the Gyazo or other screenshot URLs to direct image links
2. Replace the placeholder with an HTML `<img>` tag
3. Embed the image directly in the email draft
4. Apply responsive styling (max-width: 100%, auto height)

**Note**: Make sure your Excel file contains valid screenshot URLs (Gyazo, Imgur, or direct image links).
